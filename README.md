### Why QuickPassportPhoto?
Creating standard passport and visa photos manually via Photoshop is a tedious 15-minute process. QuickPassportPhoto reduces this to a **15-second automated flow**—with absolutely zero editing skills required.

## ✨ Core Features Designed by Krishan Kumar Awasthi

## 🧑‍💻 Running the Project Locally

To run this masterpiece locally on your machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/Dev-Krishan-Kumar-Awasthi/QuickPassportPhoto.git
   ```
2. Navigate to the project directory:
   ```bash
   cd QuickPassportPhoto
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Configure your environment variables in `.env.local`.
5. Run the development server:
   ```bash
   npm run dev
   ```
6. Open [http://localhost:3000](http://localhost:3000) with your browser to experience the magic.
